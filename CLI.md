 # 현재 디렉토리는 . 현재의 상위 디렉토리(부모폴더) .. 예시 상위 폴더로 이동시 cd ..
  기초문법
  # touch  파일생성 예시 touch git.md
  # mkdir 디렉토리 생성 예시 mkdir day01
  # ls 현재 작업중인 디렉토리 예시 ls
  # mv 파일 이동 예시 mv git.md day01/ 한글이나 띄어쓰기, 기호는 불가
  # cd 디렉토리 변경 예시 cd day01/ 
  # code 파일열기 명령어 예시 code day01/git.md
